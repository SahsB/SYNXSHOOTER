# DSDL
DSDL Final Project

#### Start Server:  
```sh
$ node index.js
```
or
```sh
$ npm start
```

#### Steps to Play with Localhost:
 >> start server  
 >> open url "localhost:3000" with browser  
 >> open mobile app and key in the magic hash value
